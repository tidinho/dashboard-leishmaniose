# Analisador de Risco de Leishmaniose

## 🦠 Sobre o Projeto
O **Analisador de Risco de Leishmaniose** é uma aplicação voltada para **identificar e avaliar fatores de risco para a leishmaniose**, utilizando dados ambientais, socioeconômicos e clínicos. O objetivo do projeto é **fornecer uma ferramenta que auxilie profissionais de saúde e pesquisadores a mapear áreas de risco e priorizar intervenções preventivas**.

---

## ⚙️ Funcionalidades
- Avaliação do risco de leishmaniose com base em múltiplos fatores.
- Importação e análise de dados geográficos e populacionais.
- Relatórios detalhados do nível de risco por região.
- Visualização de resultados em gráficos e mapas interativos.
- Possibilidade de atualização contínua conforme novos dados forem inseridos.

---

## 🛠 Tecnologias Utilizadas
- **Linguagem de programação:** Python
- **Bibliotecas:**
  - `pandas` para manipulação de dados
  - `numpy` para cálculos estatísticos
  - `matplotlib` e `seaborn` para visualizações
  - `folium` ou `geopandas` para mapas interativos
- **Banco de dados:** SQLite ou PostgreSQL
- **Framework (opcional):** Django ou Flask para interface web
